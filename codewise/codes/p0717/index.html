P0717 — Input/Turbine Speed Sensor Circuit No Signal

CAUTIONIs it safe to drive? No input speed signal at all. The transmission is flying blind on input shaft speed — harsh shifts or limp mode are likely.

What P0717 means

The TCM sees absolutely no signal from the input speed sensor. Sensor, wiring, or power/ground is dead. Many transmissions enter limp mode with no input speed signal.

Most likely causes (in order)

  1. Broken or disconnected wiring to the input speed sensor
  2. Failed input/turbine speed sensor
  3. Blown fuse feeding the sensor or TCM
  4. Damaged connector

Symptoms you might notice

  • Limp mode / stuck in one gear
  • Harsh shifts
  • No-shift condition

What to check first

Test for signal voltage and power/ground at the sensor connector with the engine running. No power = fuse or wiring; power but no signal = sensor.
